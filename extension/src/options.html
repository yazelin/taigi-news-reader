台語語音服務

請填入可信任的語音服務網址。新聞文字只會在你確認朗讀後，傳送到這個服務。

正式服務必須使用 HTTPS。本機開發可用 http://127.0.0.1:8765 或 http://localhost:8765。

請向測試管理者取得邀請碼。它只會保存在這個 Chrome 使用者設定檔，每次要求服務時只送到上方網域做身分驗證；畫面不會顯示明碼。

權限與隱私

按下儲存即表示你同意：只有在你主動要求朗讀時，擴充套件才會把你選取或擷取的新聞純文字送到上方服務，轉成台語並產生語音。它不會送出 Cookie、登入資訊、完整 HTML 或新聞網址。

建議服務會把新聞文字交給 Groq 做國語轉台語，再由伺服器上的台語 TTS 產生音訊；Groq 等供應商金鑰只放在伺服器，不會放進擴充套件。私人測試邀請碼只用於這個服務的身分驗證及用量控管，不會寫入朗讀記錄或音訊。

Chrome 只會詢問是否允許連線到你指定的服務網域；本擴充套件不會要求永久讀取所有網站。清除設定時也會刪除本機邀請碼並撤銷該服務的連線權限。
